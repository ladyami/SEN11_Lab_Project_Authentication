SEN 311 - Web Application Development

This is a lab project for SEN 311 that demonstrates the development of a secure web application using the Flask framework with MySQL database integration.

The application features user registration with password hashing, a responsive design using Bootstrap 5, and dark/light mode support with Miva University brand colors.

User authentication with secure password hashing
MySQL database integration with tbl_user table
Responsive Bootstrap 5 design with dark/light mode
Miva University color palette integration
